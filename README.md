SFPlot
=================================
SFML graph plotting library

![Sample.png](doc/Sample.png)

# Description
Graph plotting library based on SFML multimedia library.
SFPlot supports
1. Cartesian Chart (2D X-Y Chart)
2. Pie Chart
3. Rader Chart

# Requirement
* SFML library
* C++ compiler which supports C++17
